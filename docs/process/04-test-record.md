## 5. 缺陷修复记录

| 编号 | 缺陷描述 | 处理结果 |
| --- | --- | --- |
| D-01 | 详情页模板中直接调用 `application.status.name()`，在模板解析阶段存在空值调用风险 | 已改为 `currentStatus` 统一处理，并增加空值保护 |
| D-02 | 详情页模型属性使用 `application`，与 Thymeleaf 内置 application 作用域冲突，导致页面字段为空 | 已改为 `terminationApplication`，页面字段正常显示 |
| D-03 | 详情页 fragment 引用使用旧式写法 | 已改为 `~{fragments/nav :: nav('list')}` 标准写法 |
